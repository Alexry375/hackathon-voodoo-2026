## ✅ Matches well
* **Character Art:** The core unit designs (Skeleton, Red Cyclops, Green Goblin) translated well into the playable, retaining their colors and vector shapes.
* **Castle Architecture:** The general exterior appearance of the castle (grey brick pattern, distinct blue conical roofs, window placement) is visually consistent.

## ❌ Discrepancies
* **[P0] Completely incorrect core mechanic:** The playable features a deployment mechanic where the user drags units from a bottom UI deck onto empty platforms (00:05). The source game uses a drag-to-aim artillery shooting mechanic originating from the characters already stationed on the platforms (Reference `clip2_0023`, `clip2_0068`).
* **[P0] Erroneous scene swap / loss of physics context:** At 00:04 in the playable, it jarringly cuts to a perfectly upright, isolated 2D grid interior. In the source (Reference `clip2_0046`), the interior cross-section exists *inside* the actual moving, physically tilted castle taking damage on the main battlefield.
* **[P1] Invalid bottom UI:** The playable includes a large dashed container with three character unit cards at the bottom of the screen (00:04). This UI does not exist in the source game.
* **[P1] Wrong enemy bomber units:** The playable uses tiny black airplanes to drop bombs (00:01). The source game uses black birds/crows carrying bombs (Reference `clip2_0001`).
* **[P1] Flat, incorrect UI Top Bar styling:** The playable UI (00:00) uses flat colored bars and a basic rectangular "VS". The source game features heavily stylized UI with thick black strokes, a bubbly overlapping "VS", and much larger, punchy typography (Reference `clip2_0001`).
* **[P1] Explosion VFX mismatch:** The playable uses basic, flat purple colored splatters when bombs hit the castle (00:02). The source game features detailed, fiery explosions with orange/yellow blasts and grey smoke clouds (Reference `clip2_0136`).
* **[P1] Terrain detail:** The playable features completely flat, blocky brown geometry for the ground (00:00). The source has an undulating hill surface with green grass on top and textured brownish-red dirt layers below (Reference `clip2_0001`).
* **[P2] Tread wheels:** The castle treads in the playable use 4 featureless dark grey circles. The source uses 2 detailed mechanical cogs per side (Reference `clip2_0001`).
