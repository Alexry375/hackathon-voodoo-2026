## ❌ Discrepancies
* **[P0] Completely incorrect core game mechanic:** The playable ad turns the game into a "drag and drop to move units" puzzle, where the user drags a dashed path to move units between platforms (seen at 00:04 - 00:15 in Playable). The source game is clearly an aiming/artillery game where the user drags to plot a firing trajectory for character weapons (visible in Frame `clip2_0023.png` and `clip2_0068.png`). 
* **[P0] Missing combat and destruction features:** The playable ad features no actual shooting, no projectiles, no explosions, and no structural castle damage. Characters simply shift positions, and the screen fades to victory. The source game features deep combat logic including projectiles, smoke, explosions, floating damage text, and destroyed castle states (visible in Frames `clip2_0046.png`, `clip2_0136.png`, and `clip2_0158.png`).
* **[P1] Heavily downgraded character assets and missing weapons:** The characters in the playable ad are extremely generic, lack detail, and are entirely unarmed (Playable 00:04). The original source characters are highly detailed and wield massive, color-coded cannons and multi-barrel guns (Frame `clip2_0023.png`).
* **[P1] Incorrect Top UI / HUD:** The playable's UI (Playable 00:00) uses extremely thin colored bars, a tiny, basic "VS" text box, and plain unstyled black text for percentages. The original UI (Frame `clip2_0001.png`) features thick top bars, a large stylized overlapping white "Vs" logo with a heavy black outline, and large cartoon-styled percentage text with thick strokes.
* **[P1] Fabricated bottom UI panel:** The playable features a bottom UI inventory/selection bar with three character boxed icons (Playable 00:04). This UI does not exist anywhere in the source gameplay (refer to `clip2_0023.png`).
* **[P1] Severe background environment visual wipe:** The playable ad (00:00) uses flat, solid-color vector shapes for trees, sky, ground, and underground layers. The source game (Frame `clip2_0001.png`) utilizes a rich, multi-layered parallax jungle scene with varied tree shades, hills, clouds, and a detailed textured dirt boundary.
* **[P2] Simplified castle architecture:** The exterior tracks in the playable are simplified dark brown shapes lacking articulation (00:00). In the source (`clip2_0001.png`), they are detailed metal treads with moving gears and a wooden archway base. The interior wooden platforms in the playable are flat orange rectangles, missing the 3D depth and bottom support brackets seen in the game (`clip2_0023.png`).
