## ❌ Discrepancies
* **[P0] Characters missing weapons & incorrect projectile origin** — In the playable recording (00:08), characters are empty-handed, and the dotted trajectory line comes straight out of the character's body. In the source (Frame `clip2_0023.png`), each unit holds a specific, distinct weapon (blue minigun, green cannon, red cannon) with the trajectory originating from the barrel tip.
* **[P0] Extraneous UI** — The playable includes a large bottom UI tray containing three character portraits (00:07). This entire UI dock belongs to a different game loop and is completely absent in the source footage (Frame `clip2_0023.png`).
* **[P1] Missing cancel mechanic** — The source game features a distinct red "X" button next to the character while aiming, allowing the player to cancel the shot (Frame `clip2_0023.png`). This UI element and mechanic are missing in the playable (00:08).
* **[P1] Top UI styling** — The source UI features a heavily stylized, large "VS" graphic overlapping the frame, and very large percentage numbers with thick black strokes (Frame `clip2_0001.png`). The playable uses tiny, unstyled system text that is hard to read (00:00).
* **[P1] Cursor asset mismatch** — The playable uses a flat, 2D white vector cursor (00:08). The source utilizes a stylized, 3D-rendered human hand with a blue cuff (Frame `clip2_0023.png`).
* **[P1] Background & Environment art** — The source game features a detailed parallax jungle background with rolling hills, clouds, and textured grass (Frame `clip2_0001.png`). The playable replaces this with simplified, flat geometric pine trees and plain brown dirt (00:01). 
* **[P2] Base tread detailing** — The tank tracks in the source are highly detailed with visible metal gears, cogs, and tread spikes (Frame `clip2_0001.png`). The playable tracks are flat shapes lacking mechanical detail (00:00).
* **[P2] Projectile logic** — The source shows damage originating from birds dropping black bombs (Frame `clip2_0046.png`) and rockets angling down from the sky (Frame `clip2_0181.png`). The playable relies entirely on horizontal flying missiles (00:02, 00:10).
